Example Configuration

# Copy raspine-config.txt.sample to raspine-config.txt and edit
# WARNING: This file is deleted after processing for security!

# WiFi Networks (priority: higher numbers = higher priority)
wifi_ssid=HomeNetwork
wifi_password=HomePassword

wifi_ssid_2=WorkNetwork
wifi_password_2=WorkPassword

wifi_ssid_3=MobileHotspot
wifi_password_3=HotspotPassword

# Security
user_password=MySecurePassword
enable_ssh_password=false
ssh_key=ssh-rsa AAAAB3NzaC1yc2EAAAA... user@example.com

# System
hostname=my-raspine
timezone=Europe/London
wifi_country=GB

üîí Security Note: The config file is automatically deleted after processing to protect your passwords and SSH keys. Always keep a backup of your configuration settings in a secure location.

üöÄ Quick Start

1
Extract the image:
xz -d RasPINE-*.img.xz
2
Write to SD card:
sudo dd if=RasPINE-*.img of=/dev/sdX bs=4M status=progress conv=fsync
3
(Optional) Configure before boot:

Mount the boot partition and copy raspine-config.txt.sample to raspine-config.txt, then edit with your settings.

4
Boot and login:

Username: raspine

Password: raspberry (or your configured password)
