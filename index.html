🚀 Quick Start

  1. Download the latest image from above
  2. Extract the image: 
    xz -d RasPINE-*.img.xz
  3. Write to SD card: 
    sudo dd if=RasPINE-*.img of=/dev/sdX bs=4M status=progress
  4. Boot your Raspberry Pi
  5. Login via SSH or console:
    • Username: root
    • Password: raspberry
  6. Change password immediately: passwd
